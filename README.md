# DMarjar.github.io

## Sistema de Coordenadas

Contiene 4 campos:
- **x**: Coordenada x máxima.
- **y**: Coordenada y máxima.
- **Tiempo**: Tiempo de espera entre cada iteración (En segundos).
- **Cantidad**: Cantidad de puntos que serán generados por iteración.

Se tiene un botón para iniciar la generación de puntos aleatorios, y otro para detener la generación.
